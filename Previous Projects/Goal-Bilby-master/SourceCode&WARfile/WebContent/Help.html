User Manual

Goal model editor is used to create your own goal model very simply and quickly.

You can click the iuput list to edit the goal, quality, emotion, role and sub-goal. After that, you can click the add button to generate a tree structure of what you have edited. You can also import a XML file to generate a diagram. There are some buttons showing in the upper right, you can use these buttons to operate with your diagram. After saving the diagram, you can export the diagram.
