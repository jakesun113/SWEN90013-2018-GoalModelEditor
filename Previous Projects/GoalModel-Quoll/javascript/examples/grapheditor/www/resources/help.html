Graph Editor Help

In the text box on the top left hand side, input text a line per goal in the format: Please ensure that the root node only had three elements. No [parent ID] specified.

[goal type] [goal ID] [goal name] [parent ID] [visibility*]
Code for goal type
f - functional goal
e - emotional goal
q - quality goal
s - stakeholder
n - negative goal
sg - sub goal
goal ID - ID to reference the goal as a parent
goal name - Value to assign to the goal
parent ID - The parent reference ID
visibility - Optional variable to hide goal

For example, this input

f 0 "functional goal"sg 1 "sub goal" 0
s 2 stakeholder 0
e 3 "emotional goal" 0
q 4 "quality goal" 0
n 5 "negative goal" 0

will create the following goal model
